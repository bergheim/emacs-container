#+TITLE: Emacs Container

Containerized dev environment with Emacs, AI tools, and browser automation.

* Quick Start

#+begin_src bash
# Install jolo
ln -s $(pwd)/jolo.py ~/.local/bin/jolo

# Start devcontainer in any git project
cd ~/myproject && jolo

# Create worktree with isolated container
jolo --tree feature-x

# Create new project
jolo --create myapp
#+end_src

* Build

#+begin_src bash
podman build --build-arg USERNAME=$(whoami) \
             --build-arg USER_ID=$(id -u) \
             --build-arg GROUP_ID=$(id -g) \
             -t emacs-gui .
#+end_src

* What's Included

- Wolfi Linux (glibc) base
- Emacs, Neovim, tmux
- Claude Code, Gemini CLI, Codex CLI
- Language servers: Go, Rust, TypeScript, Python, Bash, YAML, Dockerfile
- Browser automation: playwright, agent-browser, webctl
- Dev tools: ripgrep, fd, jq, yq, gh, pre-commit, ruff, golangci-lint

See [[file:AGENTS.md][AGENTS.md]] for AI agent guidance.

* Networking

Ports 4000-5000 forwarded. Use ~$PORT~ env var for dev servers.
