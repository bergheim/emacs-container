#+TITLE: Emacs Container

Containerized dev environment with Emacs, AI tools, and browser automation.

* Quick Start

#+begin_src bash
# Install jolo
ln -s $(pwd)/jolo.py ~/.local/bin/jolo

# Start devcontainer in any git project
cd ~/myproject && jolo

# Create worktree with isolated container
jolo tree feature-x

# Create new project
jolo create myapp
#+end_src

* Build

#+begin_src bash
podman build --build-arg USERNAME=$(whoami) \
             --build-arg USER_ID=$(id -u) \
             --build-arg GROUP_ID=$(id -g) \
             -t emacs-gui .
#+end_src

* What's Included

Alpine Linux (musl), Emacs, AI tools (Claude/Gemini/Codex), language servers, browser automation, all the usual dev tools. See [[file:AGENTS.md][AGENTS.md]] for details.

* Worktrees

Each ~tree~ or ~spawn~ command creates an isolated git worktree with its own container. Ports auto-increment:

| Worktree | PORT |
|----------+------|
| main     | 4000 |
| tree-1   | 4001 |
| tree-2   | 4002 |

Use ~$PORT~ in dev servers. Ports 4000-5000 forwarded to host.
